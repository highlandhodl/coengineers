# UX Documentation

This folder contains user experience documentation for the CoEngineers platform.

## Contents

- [User Journeys](./user-journeys.md) - Core user flows through the platform

## Overview

The CoEngineers platform provides a 20-lab course teaching Claude Code to product managers, content creators, and knowledge workers. The UX focuses on:

- **Progressive disclosure** - Content revealed in digestible chunks
- **Zero-friction onboarding** - Start learning immediately
- **Mobile-responsive** - Works on all devices
- **Accessible** - WCAG 2.1 AA compliant

## Design Principles

1. **Simplicity** - Clean, distraction-free reading experience
2. **Consistency** - Starlight documentation patterns throughout
3. **Discoverability** - Clear navigation and search
4. **Trust** - Professional appearance, no dark patterns
