# Roadmap

This document outlines future enhancements and next steps for the CoEngineers platform.

## Current State (v1.0)

- 20-lab course structure
- Email subscription system
- Documentation website
- Comprehensive test suite
- Stakeholder documentation

---

## Short Term (Q1 2025)

### Course Launch

- [ ] Publish all 20 lab videos on YouTube
- [ ] Enable email drip campaign
- [ ] Launch community Discord/Slack
- [ ] Announce on LinkedIn and Twitter

### Platform Improvements

- [ ] Add course progress tracking (local storage)
- [ ] Implement lab completion checkmarks
- [ ] Add "Next Lab" navigation
- [ ] Create certificate of completion

---

## Medium Term (Q2-Q3 2025)

### Content Expansion

- [ ] Add advanced labs (Theme 5: Building with MCP)
- [ ] Create quick reference cheat sheets
- [ ] Record supplementary explainer videos
- [ ] Write case study blog posts

### Community Features

- [ ] User profiles (optional)
- [ ] Discussion forums per lab
- [ ] Showcase gallery of user projects
- [ ] Monthly Q&A sessions

### Analytics

- [ ] Course completion tracking
- [ ] Lab popularity metrics
- [ ] Email engagement analysis
- [ ] Conversion funnel reporting

---

## Long Term (2026+)

### Platform Evolution

- [ ] Interactive coding exercises
- [ ] AI-assisted feedback on submissions
- [ ] Team/enterprise licensing
- [ ] Certification programme

### Content Strategy

- [ ] Partner with organisations for custom courses
- [ ] Create industry-specific tracks (PM, Content, Engineering)
- [ ] Localisation to other languages
- [ ] Podcast series interviews

---

## Not Planned

Items explicitly out of scope:

- **Paid tiers** - Course remains free forever
- **Native mobile app** - Web-first approach
- **Video hosting** - YouTube handles this well
- **LMS features** - Keep it simple

---

## How to Contribute

1. Open an issue describing the enhancement
2. Reference this roadmap for context
3. Submit PR if implementing
4. Follow contributing guidelines
