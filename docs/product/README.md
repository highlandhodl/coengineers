## Product Overview

CoEngineers is a free 20-lab course teaching Claude Code to non-technical professionals. The platform was built using the spec-driven development approach it teaches.

## Success Metrics

- Course completion rate
- Email subscription growth
- User engagement (time on page, page views)
- Community growth
