## The 10 Commands You Need

| Command | What It Does | Example |
|---------|--------------|---------|
| `pwd` | Print working directory (where am I?) | `pwd` |
| `ls` | List files in current folder | `ls` |
| `cd` | Change directory | `cd Documents` |
| `mkdir` | Make a new folder | `mkdir projects` |
| `touch` | Create an empty file | `touch notes.md` |
| `cp` | Copy a file | `cp file.txt backup.txt` |
| `mv` | Move or rename a file | `mv old.txt new.txt` |
| `rm` | Delete a file | `rm unwanted.txt` |
| `cat` | Display file contents | `cat notes.md` |
| `open` | Open file in default app | `open notes.md` |

## Try It Yourself
