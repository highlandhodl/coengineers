# Prerequisites

Claude Code is the only thing you _need_ for this course. But some lessons use additional tools that make specific tasks easier.

## Required

| Tool        | What It's For             | How to Get It                                        |
| ----------- | ------------------------- | ---------------------------------------------------- |
| Claude Code | Everything in this course | [Installation guide](/getting-started/installation/) |

## Optional (But Recommended)

These tools are used in specific lessons. You can skip those lessons or find alternatives if you prefer.

### Obsidian (Week 1)

A free note-taking app that works beautifully with markdown files.

- **Used in**: Days 3, 4, 6, 7, 19
- **Cost**: Free
- **Get it**: [obsidian.md](https://obsidian.md/)

### Whisper (Week 2)

OpenAI's speech-to-text model. Can run locally for free.

- **Used in**: Days 9, 13
- **Cost**: Free (local) or pay-per-use (API)
- **Get it**: [openai/whisper on GitHub](https://github.com/openai/whisper)

### External Services (Weeks 3-4)

Some lessons use external AI services. All have free tiers.

| Service                   | Used In | Free Tier |
| ------------------------- | ------- | --------- |
| Gamma                     | Day 15  | Yes       |
| Google AI Studio (Gemini) | Day 16  | Yes       |
| ElevenLabs                | Day 17  | Limited   |
| NotebookLM                | Day 18  | Yes       |
| HeyGen                    | Day 20  | Limited   |
| Kling                     | Day 21  | Limited   |
| Canva                     | Day 24  | Yes       |
| Airtable                  | Day 26  | Yes       |

You don't need accounts for all of these right now. We'll tell you what you need at the start of each lesson.

## Don't Have Something?

No worries. Each lesson stands alone. If you don't have a particular tool:

- Skip that lesson and come back later
- Check if there's a free alternative
- Ask in the [community](/community/) for suggestions
