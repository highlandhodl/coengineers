Claude Code is the only thing you _need_ for this course. But some labs use additional tools that make specific tasks easier.

## Required

| Tool        | What It's For             | How to Get It                                        |
| ----------- | ------------------------- | ---------------------------------------------------- |
| Claude Code | Everything in this course | [Installation guide](/getting-started/installation/) |

## Strongly Recommended

### Obsidian (Themes 1-2)

A free note-taking app that works beautifully with markdown files. This course focuses on building an Obsidian-based productivity system.

- **Used in**: All of Theme 1 and Theme 2
- **Cost**: Free
- **Get it**: [obsidian.md](https://obsidian.md/)

## Optional (Theme 3-4)

These tools are used in specific labs. All have free tiers available.

### Document Skills Prerequisites

Claude Code's document skills (docx, xlsx, pptx) work out of the box. You'll need compatible viewers to open the output:

| Skill | Output Format     | Viewers                                          |
| ----- | ----------------- | ------------------------------------------------ |
| docx  | Word document     | Microsoft Word, Google Docs, LibreOffice         |
| xlsx  | Excel spreadsheet | Microsoft Excel, Google Sheets, LibreOffice      |
| pptx  | PowerPoint        | Microsoft PowerPoint, Google Slides, LibreOffice |

### External API Prerequisites (Theme 4)

Some labs integrate with external APIs. All have free tiers.

| Service                   | Lab     | Free Tier | Notes                   |
| ------------------------- | ------- | --------- | ----------------------- |
| Gamma                     | Lab 4.1 | Yes       | Presentation generation |
| Google AI Studio (Gemini) | Lab 4.2 | Yes       | Image generation        |
| ElevenLabs                | Lab 4.3 | Limited   | Text-to-speech          |

You don't need accounts for these right now. We'll tell you what you need at the start of each lab.

## Don't Have Something?

No worries. Each lab stands alone. If you don't have a particular tool:

- Skip that lab and come back later
- Check if there's a free alternative
- Ask in the [community](/community/) for suggestions
