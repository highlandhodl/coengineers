# Week 3: Creative Outputs

**Theme:** "From notes to assets that impress."

This week, you'll transform structured notes into production-ready creative briefs for presentations, images, audio, diagrams, social media, and video.

## What You'll Learn

- Generate presentation outlines with hook → journey → landing structure
- Create detailed image prompts for AI image generators
- Write scripts optimised for text-to-speech
- Build process diagrams with Mermaid syntax
- Break long-form content into engaging social posts
- Write video briefs and shot lists

## Thistle & Thorn Context

Jamie needs to create marketing materials for the Christmas campaign:

- Presentation for a potential distributor
- Social media content for Instagram and LinkedIn
- Audio for a podcast episode
- Video content for the website
- Diagrams explaining the whisky-making process
