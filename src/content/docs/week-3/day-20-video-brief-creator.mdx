## What You'll Learn

- Write scripts for AI avatar videos
- Include scene directions and timing
- Create talking head video content
- Prepare for HeyGen generation

## The Scenario

Thistle & Thorn wants video content for the website but can't afford a film crew. HeyGen avatars can deliver brand messages. Jamie needs to write the scripts.

## Key Concepts

| Concept             | Description                                       |
| ------------------- | ------------------------------------------------- |
| **Scene Structure** | Scene number, background, avatar action, dialogue |
| **Natural Speech**  | Contractions, pauses, conversational rhythm       |
| **Short Scenes**    | Under 30 seconds each for engagement              |
| **Pre-Planning**    | Detailed briefs save generation credits           |

## What You'll Create

- Video overview with title, duration, and purpose
- Avatar and voice recommendations
- Scene-by-scene script with timing
- Background suggestions and transitions

## Quick Reference

- Write for speaking - contractions, natural rhythm
- Be specific about scenes - help yourself in HeyGen
- Include timing - know your target duration
- Think visually - what's happening on screen?
