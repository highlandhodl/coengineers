## What You'll Learn

- Understand how Claude Code reads project context
- Create effective CLAUDE.md files
- Structure information for maximum AI comprehension
- Update project memory as your work evolves

## Key Concepts

| Concept               | Description                                                       |
| --------------------- | ----------------------------------------------------------------- |
| **CLAUDE.md**         | A markdown file that gives Claude Code context about your project |
| **Project Context**   | Information about goals, structure, conventions, and common tasks |
| **Persistent Memory** | Context that persists across conversations in the same project    |
| **Conventions**       | Rules and patterns Claude should follow in your project           |

## What You'll Create

- A `CLAUDE.md` file for your Obsidian vault
- Project-specific instructions for how Claude should help you
- A template you can adapt for future projects

## Real-World Applications

- Get consistent, contextual help across all your work
- Teach Claude your preferences and conventions once
- Reduce repetition when asking for help with recurring tasks
