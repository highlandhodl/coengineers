## A Real Example

**Starting point**: A 45-minute client discovery call recording.

**Commands used**:

1. `/whisper-pipeline` → Transcript
2. `/meeting-notes` → Structured summary
3. `/proposal` → Draft proposal
4. `/case-study` → Case study template (for later)

**Time**: 12 minutes total.

**Old way**: 3+ hours of manual work.
