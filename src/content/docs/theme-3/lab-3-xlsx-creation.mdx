## What You'll Learn

- Use Claude Code's xlsx skill to create spreadsheets
- Add formulas and calculations
- Apply formatting and conditional styling
- Structure data for analysis and reporting

## Key Concepts

| Concept            | Description                                         |
| ------------------ | --------------------------------------------------- |
| **xlsx Skill**     | Claude Code's built-in spreadsheet creation ability |
| **Formulas**       | SUM, AVERAGE, VLOOKUP and other Excel functions     |
| **Formatting**     | Cell styles, colours, borders, and number formats   |
| **Data Structure** | Organising information in rows, columns, and sheets |

## What You'll Create
