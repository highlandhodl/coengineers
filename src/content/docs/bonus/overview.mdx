# Bonus: The Capstone

Build and ship a real project in 3 days.

## The Challenge

You've learned 28 days of skills. Now it's time to put them all together and build something real.

Three professional frameworks guide you from idea to shipped product:

## Daily Breakdown

| Day | Framework | What You'll Do |
|-----|-----------|----------------|
| 29 | [BMAD Method](/bonus/day-29-bmad-method/) | Plan your project with Breakthrough Method for Agile AI-Driven Development |
| 30 | [PRP Framework](/bonus/day-30-prp-framework/) | Build using Prompt-Response-Product context engineering |
| 31 | [Ship It!](/bonus/day-31-ship-it/) | Polish, publish, and launch your project |

## Project Ideas

Choose something that:
1. Uses skills from multiple weeks
2. Is genuinely useful to you
3. Can be completed in 3 days

Examples:
- A personal website with blog
- A documentation site for a project
- A content automation pipeline
- Your own slash command library
- A chatbot for your business

## What You'll Build

By Day 31, you'll have:
- A complete, working project
- Professional documentation
- A published/shareable link
- Portfolio-worthy work
