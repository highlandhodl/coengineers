# Bonus: The Capstone

**Theme:** "Build and ship something real."

You've learned 28 days of skills. Now it's time to put them all together and build something of your own.

## What You'll Learn

- Apply the BMAD Method (Brief, Map, Analyse, Deliver) for complex projects
- Use the PRP Framework for advanced requirements engineering
- Ship a complete project from idea to live deployment

## Project Ideas

Choose something that:

1. Uses skills from multiple weeks
2. Is genuinely useful to you
3. Can be completed in 3 days

Examples:

- A personal website with blog
- A documentation site for a project
- A content automation pipeline
- Your own slash command library
- A chatbot for your business

## What You'll Build

By Day 31, you'll have:

- A complete, working project
- Professional documentation
- A published/shareable link
- Portfolio-worthy work
