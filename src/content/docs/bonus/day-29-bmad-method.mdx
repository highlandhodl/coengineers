## What You'll Learn

## The Scenario

You've learned 28 days of skills. Now it's time to build something real. Pick a project that uses skills from multiple weeks, is genuinely useful to you, and can be completed in 3 days.

## Key Concepts

| Concept               | Description                                 |
| --------------------- | ------------------------------------------- |
| **Breakthrough Goal** | The big outcome you're aiming for           |
| **Method**            | How you'll achieve it (Day 29, 30, 31 plan) |
| **Agents**            | What AI agents/tools help with each part    |
| **Deliverables**      | What you'll produce at the end              |

## What You'll Create

- Capstone project selection
- Breakthrough goal statement
- 3-day method plan
- Agent/tool assignments
- Deliverables list

## Quick Reference
