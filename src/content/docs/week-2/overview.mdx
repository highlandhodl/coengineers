# Week 2: Content Processing

This week, we build your content ingestion pipeline. By the end, you'll have tools to handle any input format and output consistent, structured notes.

## Daily Breakdown

| Day | Title | What You'll Learn |
|-----|-------|------------------|
| 8 | [Transcript Processor](/week-2/day-8-transcript-processor/) | Meeting and podcast transcripts to structured notes |
| 9 | [Voice Memo Structurer](/week-2/day-9-voice-memo-structurer/) | Rambling recordings to organised thoughts |
| 10 | [YouTube Summariser](/week-2/day-10-youtube-summariser/) | Video transcripts to summaries |
| 11 | [Meeting Notes Standardiser](/week-2/day-11-meeting-notes-standardiser/) | Inconsistent notes to your schema |
| 12 | [PDF Extractor](/week-2/day-12-pdf-extractor/) | PDFs to markdown |
| 13 | [Whisper Pipeline](/week-2/day-13-whisper-pipeline/) | End-to-end audio processing |
| 14 | [Web Archiver](/week-2/day-14-web-archiver/) | Web articles to clean markdown |

## Outcome

A complete content ingestion toolkit that handles transcripts, voice memos, videos, PDFs, and web pages.

Start with [Day 8: Transcript Processor](/week-2/day-8-transcript-processor/).
