## What You'll Learn

- Process raw meeting notes into structured action items
- Identify key decisions and next steps from unstructured text
- Create consistent meeting summaries with clear ownership and deadlines
- Build reusable prompts for meeting note processing

## The Scenario

Sarah recorded last week's distributor call. The raw transcript is 47 minutes of rambling. Fiona needs to know what was decided. Jamie to the rescue.

## Key Concepts

| Concept                  | Description                                                  |
| ------------------------ | ------------------------------------------------------------ |
| **Structure Extraction** | Pulling decisions, actions, and questions from chaotic notes |
| **Action Item Clarity**  | Every action needs an owner and a deadline                   |
| **Reusable Templates**   | Save your best prompts for repeated use                      |
| **Context Matters**      | Meeting type and attendees affect what to extract            |

## What You'll Create

- Clean meeting summaries with frontmatter
- Action items table with owners and deadlines
- Open questions for follow-up
- Reusable processing template

## Quick Reference

- Messier input = more valuable the processing becomes
- Always specify the structure you want in your output
- Include context about who needs to see this
- Save your best prompts as templates
