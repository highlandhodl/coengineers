## What You'll Learn

- Process raw meeting notes into structured action items
- Identify key decisions and next steps from unstructured text
- Create consistent meeting summaries with clear ownership and deadlines
- Build reusable prompts for meeting note processing

## Key Concepts

| Concept                       | Description                                                                           |
| ----------------------------- | ------------------------------------------------------------------------------------- |
| **Structured Extraction**     | Pulling specific types of information (decisions, actions, questions) from messy text |
| **Action Item Clarity**       | Ensuring every action has an owner and a deadline                                     |
| **Template-Based Processing** | Creating reusable prompts for recurring tasks                                         |
| **Multi-File Analysis**       | Processing multiple meeting notes together to spot patterns                           |

## What You'll Create

By the end of this lab, you'll have:

- Clean summaries of three different meeting types
- A reusable meeting processor template
- Understanding of how to tailor processing to meeting context

## Real-World Applications

- **Team Standups:** Quickly capture blockers and commitments
- **Client Calls:** Document feedback, requests, and next steps
- **Strategy Sessions:** Record decisions and open questions
- **Sprint Planning:** Track committed work and dependencies
- **One-on-Ones:** Note discussion topics and action items
