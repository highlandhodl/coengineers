## What You'll Learn

- Summarise long email threads into key decisions and actions
- Triage inbox backlogs by urgency and required action
- Draft professional responses based on email context
- Process multiple related emails to understand project status
- Create email templates for recurring scenarios

## Key Concepts

| Concept                | Description                                                  |
| ---------------------- | ------------------------------------------------------------ |
| **Thread Archaeology** | Extracting current status from long, messy email chains      |
| **Inbox Triage**       | Categorising emails by urgency (URGENT, ACTION, REVIEW, FYI) |
| **Response Drafting**  | Using AI to create 80% drafts, adding your 20% polish        |
| **Template Library**   | Building reusable structures for common email types          |
| **Status Extraction**  | Understanding project state from email conversations         |

## What You'll Create

By the end of this lab, you'll have:

- Clear summaries of complex email threads
- Triaged inbox with prioritised actions
- Professional email drafts ready for review
- Email templates for recurring scenarios

## Real-World Applications

- **Project Coordination:** Extract status from lengthy email threads
- **Client Communication:** Draft thoughtful responses quickly
- **Team Updates:** Triage morning inbox in minutes
- **Meeting Follow-Ups:** Template-based recap emails
- **Stakeholder Management:** Summarise decisions for executives
